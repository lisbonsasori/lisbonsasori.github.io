COFRE DE SENHAS

Como utilizadores frequentes de vários serviços disponíveis na Internet (e não só), necessitamos normalmente de credencias de acesso.
É a conta do Facebook, é a conta de e-mail, é o site das finanças, do banco, da escola... enfim um conjunto de serviços para os quais, devemos sempre ter credenciais de acesso diferentes por questões de segurança.
Com tantas senhas de acesso torna-se difícil a qualquer ser humano manter essa informação apenas na memória.
Foi nesse sentido que criei esta aplicação para gerir de forma segura todas as suas senhas de acesso.
Os dados inseridos são encriptados, ou seja, são convertidos em códigos irreconhecíveis pelo ser humano e acessíveis apenas usando uma senha de acesso definida pelo utilizador.
Os seus dados ficam verdadeiramente protegidos.
O programa vem equipado com um gerador versátil que produz diferentes tamanhos de senhas, de 6 a 36 caracteres alfanuméricos.
Em resumo, o COFRE DE SENHAS é muito fácil de utilizar, estando ao alcance do mais desajeitado utilizador de informática.
Não necessita de qualquer instalação, bastando fazer o download, descompactar para uma pasta e começar imediatamente a usar.

NUMISMÁRIO PORTUGUÊS

Este Programa destina-se a todos os colecionadores na área da numismática.
Os objetivos deste programa são a simplicidade e a intuitividade.
Permite a consulta ou inserção personalizada das moedas do colecionador.
O NUMISMÁRIO PORTUGUÊS tem uma interface com o utilizador muito simplificada e é completamente intuitivo.

Funções básicas do programa:
- Visualização fácil de toda a sua coleção de moedas;
- Registo fácil de Moedas Portuguesas;
- Registo fácil de Moedas do Mundo;
- Inserção de imagens das suas moedas que devem ser colocadas na pasta Moedas do programa;
- Alteração fácil dos dados de qualquer moeda;
- Comparação fácil dos dados de duas moedas;
- Não necessita de instalação. Pode guardar a pasta do programa no seu computador onde desejar.

FERRAMENTA PARA TESTAR A CONJECTURA DE COLLATZ

Este aplicativo foi projetado para dar-lhe cada passo da conjectura de Collatz.
Após inserir um número n e clicar no botão para calcular, o aplicativo mostrará cada passo dado até que o resultado seja 1, que é o que a conjectura diz.
É muito simples de usar e é extremamente rápido!
Ele pode manipular números grandes e o resultado será quase imediato.
Esta é uma ótima maneira de se divertir inserindo números e ver a sua sequência na Conjectura de Collatz, assim como pode ser um bom aplicativo para entender esta conjectura mais profundamente.
Tem ainda a possibilidade de salvar os dados obtidos em ficheiros de texto.
Não necessita de qualquer instalação, bastando fazer o download, descompactar para uma pasta e começar imediatamente a usar.
